# **Two Sum**

Given an array of integers **`nums`** of length **`n`** and an integer **`target`**, find **two distinct indices** **`i`** and **`j`** such that

$$
\text{nums}[i] + \text{nums}[j] = \text{target}.
$$

Return the pair of indices `[i, j]`. If no such pair exists, indicate failure.

## ğŸ¢ Bruteâ€‘Force Approach (O(nÂ²) Time)

1. **Double loop** over all pairs `(i, j)` where `i < j`.
2. **Check** if `nums[i] + nums[j] == target`.
3. **Return** `[i, j]` on the first match.

```cpp

    int n = nums.size();
    for (int i = 0; i < n; ++i) {
        for (int j = i + 1; j < n; ++j) {
            if (nums[i] + nums[j] == target)
                return {i, j};
        }
    }
    return {};  // no valid pair
}
```

## âš¡ Optimal Oneâ€‘Pass Hash Table (O(n) Time)

### **Key Idea**

Maintain a hash map **`seen`** mapping each number â†’ its index as you iterate. For each **`nums[i]`**:

1. Compute its **complement**: `complement = target - nums[i]`.
2. **If** `complement` exists in `seen`, weâ€™ve found a solution: return `[seen[complement], i]`.
3. **Else**, record `seen[nums[i]] = i` for future checks.

This way each element is processed **once**, and lookups/inserts take **O(1)** on average.

using namespace std;

/**
 * Returns indices of the two numbers that add up to target.
 */
vector<int> TwoSum(const vector<int>& nums, int target) {
    unordered_map<int,int> seen;
    seen.reserve(nums.size());

    for (int i = 0; i < nums.size(); ++i) {
        int complement = target - nums[i];
        auto it = seen.find(complement);
        if (it != seen.end()) {
            return {it->second, i};
        }
        // store index of current number
        seen[nums[i]] = i;
    }
    return {};  // no valid pair found
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n, target;
    cin >> n >> target;
    vector<int> arr(n);
    for (int i = 0; i < n; i++)
        cin >> arr[i];

    vector<int> result = TwoSum(arr, target);
    if (result.empty()) {
        cout << "No valid pair found\n";
    } else {
        cout << result[0] << " " << result[1] << "\n";
    }
    return 0;
}

## â“ Frequently Asked Questions (FAQs)

**Q1: What if there are multiple valid pairs?**

**Q2: Can we do this in two passes?**

> Yes: first build the map of all numbersâ†’indices, then for each `nums[i]`, check if `target-nums[i]` exists and is not `i`. But one pass is more efficient.
