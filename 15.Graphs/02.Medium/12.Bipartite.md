# 🎨 Is Graph Bipartite? [Leetcode #[785](https://leetcode.com/problems/is-graph-bipartite/description/)]

This is one of those problems where the idea is **super simple**, but the framing makes it feel deeper than it is.

At its core:

> Can we split the graph into **two groups** such that no edge connects nodes within the same group?

That’s it.



# 📌 Problem Statement

Given a graph (as an adjacency list), determine whether it is **bipartite**.

### Definition

A graph is bipartite if:

* its nodes can be divided into **two sets**
* **no edge connects nodes within the same set**



## 🧠 Intuition (The Real Insight)

Think of this like:

> “Can I color the graph using only 2 colors such that no two adjacent nodes have the same color?”

🎯 That’s the whole problem.

So we convert:

* Group A → color `0`
* Group B → color `1`



## 💡 Core Idea

### Try to 2-color the graph

* If you succeed → bipartite ✅
* If conflict occurs → not bipartite ❌

## 🔑 Key Insight

> A graph is bipartite **iff it has no odd-length cycle**

That’s the theory behind the code.



### ✅ What our Code Does

Our code uses:

* DFS traversal
* `color[]` array:

  * `-1` → unvisited
  * `0` → color A
  * `1` → color B

### Logic:

1. Assign a color to a node
2. Visit all neighbors
3. If neighbor is:

   * unvisited → assign opposite color
   * same color → return false (conflict)

This is the standard solution ✅


## 🐢 Brute Force Idea (Not practical)

Try all possible 2-color assignments and check validity.

### Why bad?

* exponential possibilities
* totally infeasible



## ⚡ Optimal Approach

### DFS / BFS Coloring

### Steps:

1. Initialize color array with `-1`
2. For each unvisited node:

   * start DFS
   * assign color
3. For each neighbor:

   * if unvisited → assign opposite color
   * if same color → return false



## 🧾 Our Code (Clean + Explained)

```cpp id="5o2k9y"
#include <bits/stdc++.h>
using namespace std;

class Solution {
private:
    bool dfs(int node, int col, vector<vector<int>>& graph, vector<int>& color) {
        color[node] = col;

        for (auto it : graph[node]) {
            if (color[it] == -1) {
                // assign opposite color
                if (dfs(it, !col, graph, color) == false)
                    return false;
            } 
            else if (color[it] == col) {
                // conflict: same color on adjacent nodes
                return false;
            }
        }
        return true;
    }

public:
    bool isBipartite(vector<vector<int>>& graph) {
        int n = graph.size();

        vector<int> color(n, -1);

        // handle disconnected graph
        for (int i = 0; i < n; ++i) {
            if (color[i] == -1) {
                if (dfs(i, 0, graph, color) == false)
                    return false;
            }
        }
        return true;
    }
};
```





## 📊 Complexity Analysis

### Time Complexity

### `O(V + E)`

* visit each node once
* traverse each edge once



### Space Complexity

### `O(V)`

* color array
* recursion stack (DFS)

## ❓ FAQs

### Q1. Why does odd cycle break bipartite?

Because you’ll eventually try to assign same color to adjacent nodes.



### Q2. Why use `-1` for unvisited?

To distinguish from valid colors (0 and 1)



### Q3. Can a disconnected graph be bipartite?

Yes, each component must independently satisfy the condition.



### Q4. Can we use BFS?

Yes, exact same logic applies.



### Q5. What if graph has no edges?

Always bipartite.



## ✨ BFS Version (Optional)

```cpp id="f3l7x2"
bool isBipartite(vector<vector<int>>& graph) {
    int n = graph.size();
    vector<int> color(n, -1);

    for (int i = 0; i < n; i++) {
        if (color[i] == -1) {
            queue<int> q;
            q.push(i);
            color[i] = 0;

            while (!q.empty()) {
                int node = q.front(); q.pop();

                for (auto it : graph[node]) {
                    if (color[it] == -1) {
                        color[it] = !color[node];
                        q.push(it);
                    } else if (color[it] == color[node]) {
                        return false;
                    }
                }
            }
        }
    }
    return true;
}
```
